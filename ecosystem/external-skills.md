## How to use this list

1. **Hit a wall?** Scan the relevant section before building. Solutions to common problems usually exist.
2. **Don't install everything.** Each installed plugin/skill costs tokens. Keep the active set small; archive the broader list as a reference.
3. **Read before installing.** Skills are instructions; malicious or low-quality ones can compromise your sessions. Examine the actual files before you install third-party skills.
4. **Adapt rather than adopt.** When a community skill nearly fits your use case, often the best path is to read it, understand the pattern, and write your own version that fits *your* workflow.
5. **Check for updates.** The ecosystem moved fast through 2026; some skills here may have been superseded.
